# task
Два охотника одновременно стреляют в цель. Известно, что вероятность
попадания у первого охотника равна **0,2**, а у второго – **0,6**. В результате первого залпа
оказалось одно попадание в цель. Чему равна вероятность того, что промахнулся
**первый охотник**?

Для решения этой задачи используется **формула Байеса**.
